Kantorku

Kantorku is a web application built using Laravel. This application provides functionality for employees to make some bookings based on the company availability feature. Several features are included such as booking a car, booking a meeting room and booking catering. Here also provide simple email notification for every booking process.

Teman Bunda

Teman Bunda is a mobile application built using Flutter. This application provides information related to maternity. Anyone especially female can learn one or two things about maternity. We provide the baby growth process information and several videos from relevant practitioners. Not only that, but we also provide baby knock to count your baby activity. .
